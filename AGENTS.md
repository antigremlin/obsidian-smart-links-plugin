# Contributor Guide

## Dev Environment Tips

- The runtime is Obsidian, an Electron app
- The code is built with esbuild
- Use `npm run lint` for linting with eslint
- Use `npm run format` for formatting with prettier

## Use For Reference

- Use <https://docs.obsidian.md/Home> as a guide/reference to Obsidian plugin development
- Use <https://codemirror.net/docs/> as a guide/reference to CodeMirror editor v6
- Use <https://github.com/zolrath/obsidian-auto-link-title> for inspiration
- Use <https://github.com/mdelobelle/obsidian_supercharged_links> for inspiration
